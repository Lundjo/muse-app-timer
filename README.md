## Features

- Create custom bundles of apps you want to block together
- Set specific time windows and days when apps should be blocked
- Variable timer before opening any app to encourage mindful usage
- One editing hour per day to prevent impulsive schedule changes

## Requirements

- Android 8.0 (API 26) or higher
- Accessibility Service permission
- Display over other apps permission
- Usage access permission

## How to use

1. Open the app and grant all required permissions
2. Go to Settings and set your daily editing hour
3. Tap the grid icon to create a bundle of apps
4. Tap the clock icon to set blocking schedules for your bundles

## Known Issues

- The variable timer behavior may be inconsistent across different devices and Android versions due to limitations of the Accessibility Service API
